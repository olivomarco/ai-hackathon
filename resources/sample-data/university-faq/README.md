# Northfield University FAQ Corpus

This folder contains a fictional document corpus for **Northfield University**, a tech-focused university used throughout the hackathon. The files are designed for retrieval-augmented generation (RAG) practice in later challenges.

## What is in this corpus?
The corpus includes 12 Markdown files covering common student support topics:

- Admissions
- Financial aid
- Housing and dining
- Academic programs
- Course registration
- Library services
- Student clubs
- Career services
- International student support
- IT support
- Academic integrity
- Campus health and counseling

Each document is written in an official but student-friendly tone and includes realistic details such as deadlines, office hours, service descriptions, and contact information.

## How to use it
You can use this corpus to:

- Test document ingestion pipelines
- Build chunking and indexing workflows
- Experiment with semantic search and retrieval
- Evaluate whether a University Q&A Assistant gives grounded answers

## Notes for facilitators
- Northfield University is fictional.
- Dates, policies, and contact addresses are invented for learning purposes.
- The content is intentionally broad enough to support varied student questions, including edge cases about deadlines, support services, and policy boundaries.
