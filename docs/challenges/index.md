# Challenges

Seven challenges take you from zero to a deployed AI application. The sequence is intentional: first make the environment work, then deploy and shape a model, then add orchestration, grounding, evaluation, and final integration.

<div class="challenge-card" markdown="1">

| Challenge | Focus | Estimated Time | Outcome |
|-----------|-------|----------------|---------|
| [00 — Setup & Orientation](challenge-00) | Hub + project setup | 30 min | A working Azure AI Foundry environment |
| [01 — First Model Deployment](challenge-01) | Model deployment + SDK call | 45 min | Your first live model interaction |
| [02 — Prompt Engineering](challenge-02) | Prompt quality and control | 1 hr | A stronger, more reliable system prompt |
| [03 — Prompt Flow](challenge-03) | Multi-step orchestration | 1.5 hr | A chained workflow with logic |
| [04 — RAG](challenge-04) | Grounding with your data | 1.5 hr | A retrieval-backed AI experience |
| [05 — Evaluation](challenge-05) | Metrics + responsible AI | 1 hr | Evidence for what is and is not working |
| [06 — Deploy & Integrate](challenge-06) | Endpoint deployment + app integration | 1 hr | A production-shaped application path |

</div>
